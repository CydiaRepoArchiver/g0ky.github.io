How to add Cydia Repo

  1. Open Cydia
  2. Tap the Sources tap
  3. Tap the Edit Button
  4. Tap the Add button
  5. Type: delete http and add https://g0ky.github.io
  6. Tap Add Source

Or Press "Add Cydia Repo Button"
